Northwind SQL & Power BI Analytics Project

Project Overview
This project is an in depth SQL analysis of the Northwind database.  
It focuses on business insights such as revenue, customers, products, employees, and operational performance.

The goal is to demonstrate data analysis skills using SQL for business decision making.

 Tool Used
 MYSQL
Project Structure
The project is grouped into four tiers:

 Tier 1: Basic Analysis
- Total orders, customers, products
- Revenue calculation
- Basic grouping and filtering
- Customer and product distribution

 Tier 2: Intermediate Analysis
- Revenue by product, category, employee, customer
- Monthly and yearly performance
- Discount analysis
- Customer segmentation

 Tier 3: Advanced Analysis
- RFM customer analysis (Recency, Frequency, Monetary)
- Ranking using window functions
- Cohort and retention analysis
- Growth trends and running totals
- Repeat customer behavior analysis

 Tier 4: Business Reporting
- Geographic revenue analysis (country & city)
- Shipping performance analysis
- Executive KPI dashboard
- Employee performance summary

 Key Business Insights

- Identified top performing products and categories
- Measured total revenue and discount impact
- Analyzed customer purchasing behavior and loyalty
- Ranked employees based on sales performance
- Evaluated shipping and logistics efficiency
- Segmented customers using RFM analysis

 Key Skills Demonstrated
- SQL Aggregations (SUM, COUNT, AVG)
- Joins (INNER, LEFT JOIN)
- Subqueries
- CTEs (Common Table Expressions)
- Window Functions (RANK, LAG)
   Business Data Analysis
- KPI reporting
