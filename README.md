Northwind SQL & Power BI Analytics Project

Project Overview
This project is an in depth SQL and Power BI analysis of the Northwind database.  
It focuses on business insights such as revenue, customers, products, employees, and operational performance.

The goal is to demonstrate data analysis skills using SQL and Power BI for business decision making.

 ToolS Used
 MYSQL
 POWER BI
Project Structure
The SQL project is grouped into four tiers:

 Tier 1: Basic Analysis
- Total orders, customers, products
- Revenue calculation
- Basic grouping and filtering
- Customer and product distribution

 Tier 2: Intermediate Analysis
- Revenue by product, category, employee, customer
- Monthly and yearly performance
- Discount analysis
- Customer segmentation

 Tier 3: Advanced Analysis
- RFM customer analysis (Recency, Frequency, Monetary)
- Ranking using window functions
- Cohort and retention analysis
- Growth trends and running totals
- Repeat customer behavior analysis

 Tier 4: Business Reporting
- Geographic revenue analysis (country & city)
- Shipping performance analysis
- Executive KPI dashboard
- Employee performance summary

 Key Business Insights

- Identified top performing products and categories
- Measured total revenue and discount impact
- Analyzed customer purchasing behavior and loyalty
- Ranked employees based on sales performance
- Evaluated shipping and logistics efficiency
- Segmented customers using RFM analysis

 Key Skills Demonstrated
- SQL Aggregations (SUM, COUNT, AVG)
- Joins (INNER, LEFT JOIN)
- Subqueries
- CTEs (Common Table Expressions)
- Window Functions (RANK, LAG)
   Business Data Analysis
- KPI reporting and customer analytics.

This project also includes interactive dashboards built in Power BI using the Northwind dataset.
1. Sales Executive Dashboard
   The business generated total revenue of 9.65K from 8 orders
   On-time delivery rate is high (87.5%), indicating good logistics performance
  Revenue varies across cities, with some cities contributing significantly more
  Product-level analysis shows that a few products drive most of the revenue

2. Customer and Sales Performance Dashboard
  Analyzes customer purchase behavior
  Tracks total sales and revenue trends
  Identifies top customers and regions

 3. Product and Inventory Dashboard
   Monitors stock levels and product performance
   Identifies fast moving and slow moving products
   Helps in inventory optimization decisions

  4. Operational and Logistics Dashboard
   Analyzes order processing and shipping performance
   Tracks delivery timelines and efficiency
    Identifies operational bottlenecks
